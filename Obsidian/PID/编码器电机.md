### 当顺时针为正时
TIM_EncoderInterfaceConfig(TIM3, TIM_EncoderMode_TI12, TIM_ICPolarity_Rising, TIM_ICPolarity_Falling)
顺时针旋转 → 编码器计数增加 → Actual 值为正
逆时针旋转 → 编码器计数减少 → Actual 值为负

if (PWM >= 0)
{
    GPIO_SetBits(GPIOB, GPIO_Pin_13);  // 正转方向
    GPIO_ResetBits(GPIOB, GPIO_Pin_12);
    PWM_SetCompare1(PWM);
}
else
{
    GPIO_ResetBits(GPIOB, GPIO_Pin_13);  // 反转方向  
    GPIO_SetBits(GPIOB, GPIO_Pin_12);
    PWM_SetCompare1(-PWM);
}

### 当逆时针为正时
TIM_EncoderInterfaceConfig(TIM3, TIM_EncoderMode_TI12, TIM_ICPolarity_Rising, TIM_ICPolarity_Rising)
逆时针旋转 → 编码器计数增加 → Actual 值为正
